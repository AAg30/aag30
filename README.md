# AAg30 - Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my AI projects and professional information.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with gradient backgrounds
- **Interactive Elements**: Smooth animations and hover effects
- **Navigation**: Sticky navbar with smooth scrolling
- **Sections Included**:
  - Hero section with call-to-action
  - About me with skills and stats
  - Featured projects
  - Contact and social links
  - Footer

## 📁 Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive functionality and animations

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Or deploy to GitHub Pages, Netlify, or any web hosting service

## 📝 Customization

### Update Personal Information
Edit the following in `index.html`:
- Hero title and tagline
- About me content
- Skills list
- Project descriptions
- Social media links
- Email address

### Color Scheme
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #0066ff;
    --secondary-color: #00d4ff;
    --dark-bg: #0a0e27;
    /* ... other colors ... */
}
```

### Add More Projects
Add new project cards to the projects section:
```html
<div class="project-card">
    <h3>Project Name</h3>
    <p>Project description</p>
    <a href="#" class="project-link">View Project</a>
</div>
```
